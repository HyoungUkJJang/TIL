자바 프로그래밍에서 NullPointerException을 보게된다.
null을 리턴하기 때문 or null 체크를 안했기 때문에 발생하는 에러이다.

메소드에서 작업중 특별한 상황에서 값을 제대로 리턴할 수 없는 경우 선택할 수 있는 방법
1. 예외를 던져준다
2. null을 리턴한다.
3. Optional을 리턴한다.

Optional - 오직 값 하나가 들어있을 수도 없을수도 있는 컨테이너.

주의점
1. 리턴값으로만 쓰는것을 권장함
2. 옵셔널을 리턴하는 메소드에서 null을 리턴하지 말자
3. 프리미티브 타입용 옵셔널은 따로있다.
4. 컬렉션, 맵, 스트림 어레이, 옵셔널은 옵셔널로 감싸지 않는다.
